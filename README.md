# QR/Barcode Generator API

An API to generate QR codes and barcodes from given payloads.

> Please use Node 22 or higher.

## Installation

Before installing please install the libraries listed here according to your OS:

<https://github.com/Automattic/node-canvas?tab=readme-ov-file#compiling>

Then run:

```bash
npm install
```

## Usage

```bash
npm run dev
```

## Testing

```bash
npm run test
```
